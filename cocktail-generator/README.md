# cocktail-generator
JS/XML assignment for HTTP5222

A cocktail recipe finder built with React and Vite. Search by ingredient and instantly
get matching cocktails, view full recipes, and a weird GIF for each drink j4f.

---

## API LIST

1. TheCocktailDB: Free, crowd-sourced database of drinks and cocktails. This 
API provides detailed recipes, ingredient lists, and images.  
2. GIPHY API: Large library of animated GIFs. When a user is viewing a cocktail 
recipe page, a call will be made to the API to search for and display a 
relevant GIF.  

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| React + Vite | Frontend framework & build tool |
| CocktailDB API | Cocktail data & recipes |
| GIPHY API | Animated GIFs per cocktail |
| CSS (custom) | Styling with CSS variables |
